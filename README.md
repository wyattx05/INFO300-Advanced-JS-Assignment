## 4 JavaScript Programs showcasing my skills

### PrimeGaps.js
Detects all the prime numbers between 0 and positive number, the largest gap between prime numbers, the prime numbers between that gap, and the average size of a gap.

### FibonacciFilter.js
Take input number, generates Fibonacci sequence, filters out only odd Fibonacci numbers.

### LetterFrequency.js
Tells you how often a letter is repeated in a string and tells you the first letter that is repeated.
